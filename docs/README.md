# Debris Deputy

A Real-Time AI Surveillance System for Garbage Detection and Monitoring.

## Features
- Real-time webcam monitoring
- Garbage detection using AI (YOLOv8)
- Snapshot storage with timestamp
- User feedback system
- 48-hour tracking and alerts
- Duplicate detection prevention

## Tech Stack
- Python (Flask)
- OpenCV + YOLOv8
- MongoDB
- Frontend (HTML/CSS/JS)
- Telegram API (alerts)

## Team Roles
- AI & Core System
- Backend
- Frontend
- Database
